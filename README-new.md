# Chris Bigelow's Personal Website

A modern, responsive personal website built with vanilla HTML, CSS, and JavaScript.

## Features

- **Modern Design**: Clean, gradient-based design with smooth animations
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Todo List**: Check off completed items with local storage persistence
- **Dark Mode Toggle**: Switch between light and dark themes
- **Smooth Scrolling**: Elegant navigation between sections
- **Typing Animation**: Dynamic text effects for enhanced user experience
- **Accessibility**: Keyboard navigation and focus management
- **Performance**: Lightweight and fast loading

## File Structure

```
/
├── index.html          # Main HTML file
├── styles.css          # All CSS styling
├── script.js           # JavaScript functionality
├── README-new.md       # This file
└── CNAME              # Domain configuration (preserved from original)
```

## Key Features

### Interactive Todo List
- Checkboxes that persist state in localStorage
- Visual feedback when items are completed
- Smooth animations and hover effects

### Modern UI Elements
- Gradient backgrounds and glass-morphism effects
- Smooth transitions and micro-interactions
- Responsive grid layouts
- Custom scrollbar styling

### Enhanced User Experience
- Back-to-top button that appears on scroll
- Theme toggle (light/dark mode)
- Typing animation on the intro text
- Scroll-triggered animations for content sections

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Performance

- No external dependencies (except Google Fonts)
- Optimized CSS with minimal reflows
- Efficient JavaScript with event delegation
- Small file sizes for fast loading

## Customization

The website is easily customizable:

1. **Colors**: Modify the CSS custom properties in `styles.css`
2. **Content**: Update the HTML structure in `index.html`
3. **Functionality**: Add new features in `script.js`

## Deployment

Simply upload the files to any web server or static hosting service:
- GitHub Pages
- Netlify
- Vercel
- Any traditional web hosting

The website is completely self-contained and doesn't require any build process or server-side rendering.
